# Titanic Mini Exploratory Data Analysis (EDA)

## Objective
Perform Exploratory Data Analysis on the Titanic dataset using Python.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Tasks Performed
- Data Cleaning
- Missing Value Imputation
- Feature Engineering
- Survival Analysis
- Data Visualization
- Insights Generation

## Visualizations
- Age Distribution Histogram
- Correlation Heatmap
- Survival by Family Size
- Survival by Age Group
- Survival by Embarkation Port

## Dataset
Titanic Dataset
